vim 的变量。vim设置，输出变量。获得当前文件名
可以这样 

```vimshell
vim 中定义变量要用 

let a=123  空格并不能影响效果，这一点和shell并不一样.
echo a
这样就可以看到效果。

在 vimrc 中可以写简单函数，来做一些想做的事情，比如
根据文件后缀，自动执行脚本程序。

:echo %    这个是获得当前文件的全名，包括后缀的。
:echo %<   和下面的一行作用一样。
:echo %:r  这个是获取当前文件的文件名，除去扩展后缀
```
