#### margin

margin: 的4个参数。up ,right, down , left;
如果只有一个参数，那么表示4个都是n px。可以是1--4个参数。

#### some input type

radio, checkbox , button, 他们都有对应的时间，可以在里面添加对应的 函数。可以带参数的。
checkbox 也可以写成数组。
name='chk[]'  $_REQUEST ['chk'] 取值的方式是这样的。
