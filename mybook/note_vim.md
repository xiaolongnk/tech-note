```vimshell
在 vimrc 中可以写简单函数，来做一些想做的事情，比如
根据文件后缀，自动执行脚本程序。

:echo %    这个是获得当前文件的全名，包括后缀的。
:echo %<   和下面的一行作用一样。
:echo %:r  这个是获取当前文件的文件名，除去扩展后缀
```
